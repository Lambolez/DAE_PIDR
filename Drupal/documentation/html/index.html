The DAE Platform Project

What is it About ?

This project defines a set of Drupal modules to operate on a database storing and handling document analysis resources. It is currently implemented and running on the DAE Project Server.

The latest version of this software is available at http://sourceforge.net/projects/daeplatform/

More global information and documentation is available at http://sourceforge.net/apps/mediawiki/daeplatform/index.php?title=Main_Page

This documentation is only targeted toward technical aspects of the code.
