algorithms_get_run_info ( )

Responds to AJAX calls from My Runs page for information on a given Run.

This function requires the global variable $spool to be set to the appropriate directory and takes the assumption that all negative statuses are failures.

If the run is complete and the files have not already been moved, the function moves files from spool to the user's runs directory when it calls algorithm_run_info_done()
