Detailed Description

The algorithms module allows users to view and run algorithms.

Author:
Mike Kot mdk31.nosp@m.2@le.nosp@m.high..nosp@m.edu
Version:
1.0

This module is soon to be completely rewritten. It is currently based on the fact that algorithms are hosted and executed on a specific server. This assumption does not hold anymore since the projects architecture has evolved to a web-service oriented paradigm.

Although still operational, this module is to be considered as obsolete.
