home_directory_create_run_zip ( $  run)

Create a zip file containing all result files of a given run.

This function requires the global variable $runs to be set to the appropriate directory and takes the assumption that $runs/$run/output is a valid and writeable path.

If the zip file is created, the function updates the user's file storage space usage.
