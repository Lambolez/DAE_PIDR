Detailed Description

Provides the Drupal hooks for module daedatabase.

Author:
Bart Lamiroy lamir.nosp@m.oy@c.nosp@m.se.le.nosp@m.high.nosp@m..edu
Version:
1.0

This module is aiming at making interaction with various DBMSs easier from a PHP coding point of view. It provides a set of classes and functions that allow simple SQL interaction with any DBMS in a transparent way, so that the PHP programmer can focus on SQL, rather than on the underlying DBMS syntax for connecting and querying.

This module is currently implemented for Oracle and MySQL.

Actual code and data structures are declared in daedatabase_db.inc.

daedatabase_menu ( )

Default Drupal menu hook function.

This function declares the Drupal menu names and hierarchy, as well as the associated callback functions. Since this module only provides configuration variables and DBMS interaction APIs there are no browsable contents pages or call back functions that need to be declared.

The callback functions used in this module are:

  • daedatabase_admin() for the Administration menu,
