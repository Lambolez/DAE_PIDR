invokeWSDL ( $  id  ) 

This PHP page genereates WSDL files for different algorithms with different inputs and outputs. Invokes the WSDL server for the algorithm given by $id. If $id is in the form of '<id>?wsdl' it's WSDL interface description is returned. If not, it is supposed to be combined with an HTTP_POST upload of the required input data.

Parameters:
[in] $id contains at least the algorithm id to invoke, possibly followed by '?wsdl'
