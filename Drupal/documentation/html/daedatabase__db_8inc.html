Detailed Description

Provides access to Oracle and MySQL database functions and data structures in a transparent way.

Author:
Bart Lamiroy lamir.nosp@m.oy@c.nosp@m.se.le.nosp@m.high.nosp@m..edu
Mike Kot mdk31.nosp@m.2@le.nosp@m.high..nosp@m.edu
Version:
1.0

The most important part of this module is the DAEDatabase class. This is the main API that allows a programmer to take advantage of any DBMS without having to change his or her code. This class is further derived into sub-classes that actually implement the core access functions to specific DBMSs like Oracle and MySQL.
