Detailed Description

The data repository module allows users to interact with the database using raw SQL.

Author:
Bart Lamiroy lamir.nosp@m.oy@c.nosp@m.se.le.nosp@m.high.nosp@m..edu
Version:
1.0

This module creates a Drupal form that allows any kind of SQL query to be typed in and sent to the backend database server, as defined by the Drupal configuration variables in in daedatabase.module.

data_repository_menu ( )

Default Drupal menu hook function.

This function declares the Drupal menu names and hierarchy, as well as the associated callback functions.

The call back functions used in this module are:

  • data_repository_query() for the Repository menu,
  • data_repository_display() for the Repository/SQL sub-menu.
Returns:
Drupal menu declaration array structure
